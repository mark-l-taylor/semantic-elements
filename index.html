What are semantic elements?

In HTML5 semantic elements are used to provide context to the element. These elements aid the developer and browser in understanding how the page will be rendered. The use of semantic elements can be used to help outline the content of a webpage. They can also assist with the searchability of the content.

Semantic Elements are preferred over the use of div or other simpler less descriptive elements.

Semantic elements = elements with a meaning.
